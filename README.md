# Dice iOS App

This is a simple one-page iOS app written in Swift 5 and designed with Storyboard. The app is mainly developed to practiced:
- Array
- randomElement()
- IBAction
- Auto-Layout 
- Responsive UIs

The app designed to run on all simulators.
